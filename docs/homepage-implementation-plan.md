# Lawbridge Homepage Implementation Plan

## Objective

Create the first public website page for Lawbridge Partners with a professional full-service litigation firm feel.

## Brand Rules

- Use "Lawbridge" as one word.
- Use `lawbridge_logo.png` exactly as provided; do not rearrange the bridge, laurel, wordmark, or tagline.
- Keep the top navigation visible while scrolling.
- Keep the visual language serious, institutional, and court-facing.

## Implementation Schema

```text
homepage
  sticky header
    exact logo image
    navigation links
    contact button
  hero
    firm name and positioning
    primary action
    secondary action
    practice/forum summary panel
  credibility band
    courts and tribunals served
  about section
    boutique litigation firm positioning
    service principles
  practice areas
    six core service groups
  partners
    founding partner summaries by department
  contact section
    address, email, phone numbers
```
